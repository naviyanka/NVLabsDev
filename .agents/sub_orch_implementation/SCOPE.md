# Scope: Implementation Track

## Architecture
- Theme Tokens: Generated by Stitch MCP and integrated into Tailwind/CSS.
- UI Layout: React components in `src/Nexus.Frontend/src/` styled conditionally using the `data-theme` attribute.
- Settings: API endpoints in `src/Nexus.Gateway/` and setting forms in `src/Nexus.Frontend/src/routes/settings.tsx`.
